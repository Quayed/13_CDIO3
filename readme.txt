How to configure and run the program:

The GWT SDK we use is 2.6.1.
If the project has changed to 2.6.0 you need to set the right version.

1. Run the database.sql script in an empty MySql database.
2. Fill out the Constant.java with information to connect to the database.
3. Start the database
4. Run project as Web Application (Super Dev Mode)
